Native CSS Scroll-Driven Animation (A11y)

Reduced motion + skip link + safe fallback
(모션 감소 + 스킵 링크 + 안전한 폴백)

The top bar grows with scroll progress. No JS needed in supported browsers.
(상단 바가 스크롤 진행에 따라 늘어납니다. 지원 브라우저는 JS 없이 동작.)

In unsupported browsers, it stays full as a safe fallback. This keeps content readable.
(미지원 브라우저는 가득 채워 폴백. 읽기 쉬움 유지.)
