What CSS Can Do

  • Map animation progress to scroll position
  • Reverse keyframe playback order
  • Define scroll ranges with animation-range
  • GPU-accelerated smooth animations

What CSS Cannot Do

  • Detect if user is scrolling up or down
  • Trigger animations only on specific direction
  • Remember previous scroll position
  • Conditional logic based on direction

JavaScript Solutions

  • Track lastScrollY for direction
  • Add class like .scroll-up to body
  • Use IO + direction for efficient detection
  • See IO Reverse or rAF Reverse
